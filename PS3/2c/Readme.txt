###files###
1. getTree.py
   generate large number of nodes tree, nothing related to main code

2. treeToPruferCode.py
   generate prufer code with given tree
   ###how to run this code###
   --------------------------
   python treeToPruferCode.py tree0.txt
   (argument tree0.txt can replace to tree1.txt~tree9.txt)

3. tree0.txt~tree9.txt
   10 example trees(tree9.txt has 100 nodes as required!)

4. tree0_output.txt~tree9_output.txt
   prufer code for 10 example trees

5. pruferSeqToTree.py
   construct tree given a prufer sequence
   ###how to run this code###
   --------------------------
   python pruferSeqToTree.py pru0.txt
   (argument pru0.txt can replace to pru1.txt~pru9.txt)

6. pru0.txt~pru9.txt
   10 example prufer codes(pru9.txt has 100 nodes as required!)

7. pru0_output.txt~pru9_output.txt
   trees for 10 example prufer codes with python dictionary representation
